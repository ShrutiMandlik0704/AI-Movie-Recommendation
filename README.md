# CineMatch AI - Movie Recommendation 🍿

A sleek, modern AI-powered movie recommendation web application built with **Streamlit** and Python. Designed to provide extremely accurate, graph-based recommendations using Uniform Cost Search (UCS) on keyword and genre similarities, fetching high-quality official movie posters directly from TMDB!

## Features 🚀
- **Graph Algorithm Backend:** Computes shared weights utilizing movie keywords and genres to supply perfectly localized recommendations.
- **Vibrant & Modern UI:** A fully styled Streamlit interface enforcing standard, crisp 2:3 aspect ratio movie boards.
- **TMDB Integration:** Instantly retrieves gorgeous, official high-resolution posters via The Movie Database API natively.

<br>

## How to Run Locally 💻

1. **Install Requirements:** Make sure you have python installed. Then run the following in your terminal:
   ```powershell
   pip install -r requirements.txt
   ```

2. **Start the Application:**
   ```powershell
   python -m streamlit run app.py
   ```

3. **View the Website:** Once the server finishes starting, you can view your localized site cleanly at:
   👉 **http://localhost:8501**
